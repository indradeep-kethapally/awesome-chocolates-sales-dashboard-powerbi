# Awesome Chocolates - Sales Analytics Dashboard

Interactive Power BI dashboard analyzing sales performance, shipments, 
costs, and profitability across regions and sales representatives.

## Key Features
- KPI cards: Total Sales, Boxes, Shipments, Costs, Profit, Profit % gauge
- Sales trend by month
- Shipment analysis distribution
- Sales rep performance table with profit % and LBS % indicators
- Regional breakdown (Australia, Canada, India, UK, USA, etc.)

## Tools used
Power BI Desktop, DAX, Power Query

## Data Source
Dataset provided as part of Chandoo.org's Power BI tutorial series 
(used for learning and portfolio purposes). 
Tutorial reference: https://youtu.be/ooJO7NW4uJU
